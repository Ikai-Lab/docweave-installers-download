## ⚠️ Installation Notes

### Windows
You may see a security warning:
- Click **More info**
- Then click **Run anyway**

---

### macOS
If the app is blocked:
- Right-click the app → **Open**
- Click **Allow**

---

### Linux
Run the following:

```bash
chmod +x docweave.AppImage
./docweave.AppImage
